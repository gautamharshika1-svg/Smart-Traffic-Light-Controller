## Overview

This project implements a two-way intelligent traffic light control system using Arduino Uno. The system automates traffic signal sequencing and displays a real-time countdown timer on a 7-segment display, simulating an embedded traffic management solution.

## Working Principle

1. The controller initializes the traffic sequence.
2. Green signal is activated for one direction while the opposite remains red.
3. Countdown timer displays the remaining signal duration.
4. The system transitions through yellow before switching directions.
5. The cycle repeats automatically for continuous traffic regulation.
